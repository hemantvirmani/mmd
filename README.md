# Mermaid Editor

Client-side Mermaid editor with:
- Split layout (source + live preview)
- Load from disk (`.mmd`, `.mermaid`)
- Save to disk
- localStorage persistence
- Shareable URL with base64-encoded Mermaid source in hash

## Run locally
Open `index.html` in a browser.

## Hosted URL
https://h.virmani.cc/mmd/

## Notes
- Fully client-side. No backend and no GitHub integration.
- Mermaid library is loaded from pinned CDN version `10.9.1`.
